## ğŸ”§ Installation & Setup
1. Clone the repository:
   ```sh
   git clone https://github.com/Vinit-Pothuraju/Nike-Web.git
   ```
2. Navigate to the project folder:
   ```sh
   cd Nike-Web
   ```
3. Install dependencies:

   ```sh
   npm run dev
   ```
